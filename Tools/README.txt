1. 配置本地信息（第一次）
git config --global user.name "Your Name"
git config --global user.email "email@example.com"

2. 克隆远程仓库到本地
git clone https://github.com/nuaamsc/ProjectYugong.git

3. 创建新分支
git branch (你想要的名字)
git checkout (上述分支)

4. 修改文件

5. 保存并上传
git checkout master
git merge (分支名)
git add -A
git commit -m "(给本次提交起个名)"

git remote add origin https://github.com/nuaamsc/ProjectYugong.git
git push -u origin master


脚本说明：在脚本上右键，选择使用 powershell 打开
